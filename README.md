# Сервис подготовки аттестационных листов обучающихся

## Описание

Сервис автоматизирует формирование аттестационных листов на основе учебных документов (справок об обучении и учебных планов). Он решает проблему больших объёмов ручной обработки документации и снижает вероятность ошибок.

## Как работает

Сервис состоит из трёх основных модулей:

### 1. Извлечение текста из PDF

Используется OCR-библиотека **Tesseract** (через Python-обёртку `pytesseract`). Модуль распознаёт текст даже в сканированных документах и изображениях, встроенных в PDF.

### 2. Очистка и нормализация текста

Извлечённый текст приводится к единому формату: удаляются лишние символы, исправляются типичные OCR-ошибки, выделяются ключевые поля (ФИО, дисциплины, оценки, периоды обучения).

### 3. Семантический анализ и сопоставление данных

Нормализованный текст передаётся в модель семантического анализа, которая сопоставляет распознанные фрагменты с ожидаемой структурой аттестационного листа. Для хранения и поиска эталонных эмбеддингов используется векторная база данных **ChromaDB**.

# Архитектурное решение (ADR)

## Название

Сервис для подготовки аттестационных листов обучающихся

## Статус

Решение принято, дорабатывается

## Контекст

1. Большие объемы учебной документации.
2. Необходимость автоматизации и снижение ошибок.

## Решение

Реализован модуль извлечения текста из PDF-документов с использованием OCR-технологий.  
Разработан модуль очистки и нормализации извлечённого текста.  
Подобрана модель семантического анализа текста для сопоставления извлечённых данных.  
Проведено тестирование алгоритма на различных справках об обучении.

## Последствия

Решение упрощает работу следующих подразделений: деканаты, кафедры и УМУ.  
Сокращает время и вероятность ошибок.

## Альтернативные решения

> Нет прямых альтернатив всему конвейеру, но мы можем разбить его на несколько частей и сравнить аналоги.  
> Использовано: OCR `pytesseract` и векторная БД `ChromaDB`.

### Альтернативы для OCR

- **EasyOCR** — библиотека OCR на основе глубоких сетей, с хорошей поддержкой разных языков и более гибким качеством распознавания по сравнению с классическим Tesseract.
- **PaddleOCR** — современная OCR-модель с GPU-ускорением, может давать лучший результат на сложных изображениях (например, с низким качеством, «шумом» и разномасштабным текстом).
- **Коммерческие OCR API** (Google Vision, Microsoft Azure OCR, AWS Textract) — облачные сервисы с высокой аккуратностью и поддержкой сложных макетов PDF.

### Альтернативы для векторной базы данных

- **Qdrant** — высокопроизводительная open-source векторная база, подходящая для больших объемов эмбеддингов и гибких фильтров.
- **Pinecone** — облачная managed vector DB с автоматическим масштабированием и низкими задержками.
- **pgvector (Postgres)** — расширение для PostgreSQL, которое добавляет векторный поиск прямо в реляционную БД (если инфраструктура уже использует Postgres).
